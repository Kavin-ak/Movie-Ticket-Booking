# Movie-Ticket-Booking
I've spearheaded a project aimed at developing a robust backend system for a movie ticket booking platform. The system facilitates movie searches, ticket booking, payment processing, and enables users to access their booking history seamlessly.

## Features
* Movie Browsing: Users are provided with a comprehensive list of available movies for browsing.
* Search and Filtering Capabilities: Users can employ advanced search and filtering functionalities to locate specific movies based on criteria such as title, date, location, genre, or movie ID.
* Detailed Movie Information: Detailed information pertaining to each movie, encompassing title, director, synopsis, genre, screening date, venue, total seat capacity, available seats, and ticket pricing, is readily accessible to users.
* Ticket Reservation System: Users can seamlessly reserve tickets for desired movies by specifying the movie ID, desired quantity of tickets, and corresponding total price.
* Booking History Tracking: A comprehensive booking history feature enables users to review past transactions, encompassing essential details such as movie title, director, synopsis, genre, screening date, venue, booked ticket quantity, and total expenditure.

* Movie Management: Administrators possess the capability to manage the movie repository effectively by facilitating tasks such as adding new movie entries, removing obsolete listings, and updating existing movie information.
